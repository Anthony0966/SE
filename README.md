# 클라우드 파일 공유 시스템 (미니 드라이브)

사내 협업 환경 개선을 위한 클라우드 스토리지 시스템 개발 프로젝트임. 기존 상용 클라우드 서비스의 파일의 최신버전 관리및 검색지연의 한계, 보안 한계등 문제를 해결하기 위해, 요구한 파일 공유 기능에 집중한 경량화 아키텍처를 설계한다.
